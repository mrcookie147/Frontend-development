Welcome to Frontend Development

Build interactive and engaging web experiences with frontend development. Learn the fundamentals of HTML, CSS, and JavaScript to bring your ideas to life.

HTML: The Foundation of the Web

HTML (HyperText Markup Language) is the core language used to structure web pages. It defines the content and layout of a website, including text, images, and interactive elements. HTML uses tags to create elements, such as headings, paragraphs, and lists. These elements form the basic building blocks of a website, allowing developers to arrange and organize content logically.

Basic Tags

Common tags include <h1> for headings, <p> for paragraphs, <img> for images, and <a> for hyperlinks.

Document Structure

A typical HTML document starts with a <html> tag and includes a <head> section for metadata and a <body> section for visible content.

Examples

Simple HTML code like <h1>Welcome!</h1> will create a heading on a web page. <p>This is a paragraph.</p> will create a paragraph of text.

CSS: Styling Your Website

CSS (Cascading Style Sheets) is used to style the visual presentation of web pages. It controls elements like colors, fonts, sizes, and positioning. By applying CSS rules to HTML elements, developers can create visually appealing and user-friendly web interfaces.

Selectors target specific HTML elements. Examples include tag selectors (like <p>), class selectors (.my-class), and ID selectors (#my-id).

Properties define the style attributes of an element. Common properties include color, font-size, background-color, and margin.

The CSS rule `p { color: blue; }` sets all paragraph text to blue. `h1 { font- size: 30px; }` sets all h1 headings to 30 pixels in size.

JavaScript: Bringing Web Pages to Life

JavaScript is a powerful programming language that adds interactivity and dynamic behavior to web pages. It allows developers to create animations, handle user input, and manipulate content on the fly, making websites more engaging and responsive.

With JavaScript, you can change the text of an element on a page. For instance, the code document.querySelector('h1').innerText = 'Hello, world!'; will replace the text in the <h1> heading with "Hello, world!".
